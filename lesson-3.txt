1. Giải thích các câu lệnh:

- `git commit --amend`: Sửa đổi commit cuối cùng (thêm file hoặc sửa message)
- `git commit --amend -m "<message>"`: Sửa message của commit cuối cùng
- `git restore --staged <file>`: Đưa file từ staging area về lại working directory
- `git reset HEAD~1`: Xóa commit cuối cùng, đưa các thay đổi về working directory

Tình huống ví dụ:
Bạn đánh máy nhanh và bị lỗi chính tả

```
git commit -m "Fix navigtion bug"
```

# Sửa lại commit message sai chính tả
```
git commit --amend -m "Fix navigation bug"
```
